#libcstl

## aboutme
一个纯正的C库，试图把C++里面的精华STL引入到C里面，从此以后用C写代码再也不用手写链表、堆栈了。

## new work
我不喜欢原作者用autocofig这一套，所以希望重新组织一下。

我决定用cmake重新制作libcstl的编译和安装脚本，同事进一步增进libcstl的可用性。

目前主要的问题有两个：

1. 适配macOS
2. 编译后推送到ubuntu的源里

欢迎拍砖。
